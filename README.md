# dastrass — Flutter (мобильное приложение)

Тот же бренд и навигация, что у **мобильной** версии React-сайта: нижняя «таблетка» с 5 пунктами (Главная · Избранное · Добавить · Сообщения · Профиль), тёмная тема, `#005BFE`, API как у веб-фронта.

## Запуск

```bash
cd app
flutter pub get
flutter run              # выберет устройство, или:
flutter run -d windows   # десктоп Windows
flutter run -d chrome    # браузер
```

Если раньше было «No supported devices» — в проект добавлены платформы **windows** и **web** (помимо android/ios).

По умолчанию API: в **release** без `API_BASE` — `https://api.dastrass.com/api`; в **debug** без `API_BASE` — локальный `http://127.0.0.1:8000/api` (см. `lib/config/api_config.dart`).

### Локальный Django

- **Android emulator:** можно `http://localhost:8000/api` — в приложении для Android `localhost`/`127.0.0.1` подменяются на **`10.0.2.2`** (хост вашего ПК). Явно: `--dart-define=API_BASE=http://10.0.2.2:8000/api` тоже работает.
- **Физический телефон Android:** укажите **LAN-IP** компьютера в Wi‑Fi, например `http://192.168.1.10:8000/api` (не `localhost`).
- **iOS Simulator:** `flutter run --dart-define=API_BASE=http://127.0.0.1:8000/api`

Для HTTP (не HTTPS) на Android может понадобиться `android:usesCleartextTraffic="true"` в манифесте (только для разработки).

### Чёрный экран в Chrome (Flutter web)

1. Откройте **DevTools → Console** — часто видно ошибку (CanvasKit, CORS, скрипт).
2. Если сеть блокирует CDN Google (**CanvasKit**), запустите без CDN:

```bash
flutter run -d chrome --no-web-resources-cdn
```

3. Интерфейс тёмный (`#0b1220`) — это **не баг**, пока видны заголовок и список. Пока движок грузится, на странице показывается «Загрузка…».
4. Запросы к API с `localhost` в браузере могут упираться в **CORS** — тогда данные не придут, но приложение всё равно должно показать экран (и текст ошибки у запросов). Для проверки API удобнее **Windows** или **Android**.

На **веб** в теме не подгружаются шрифты с **fonts.gstatic.com** (используются системные), чтобы не зависеть от внешней сети.
